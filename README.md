# Vue Pikaday

VueJS wrapper component for Pikaday datepicker

## Install
```bash
npm i -D @enrian/vue-pikaday
```
or
```bash
yarn add -D @enrian/vue-pikaday
```

## Usage

Please consult [documentation](https://enrian.github.io/vue-pikaday).
